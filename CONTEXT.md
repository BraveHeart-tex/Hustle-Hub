# Hustle Hub

Hustle Hub brings daily engineering work from GitLab, Jira, notes, alerts, and deployment pages into a browser-extension workspace.

## Language

**GitLab MR page state**:
A coherent view of the identity, metadata, discussions, and host appearance authored by GitLab for the currently displayed merge request. It excludes Hustle Hub decisions about Jira workflows, current-user policy, review prompts, and Claude launches.
_Avoid_: Page data, scraped DOM state
